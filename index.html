My Projects

Mainly Java simulations and interesting algorithms or demonstrations;

Slime Mold Simulation

https://github.com/Teppi-txt/Slime-Mold-Simulation

The behavior of real-life slime molds inspires a computational approach known as a slime mold simulation. In this simulation, artificial agents are guided by simple rules and are influenced by the trails of chemical markers (pheromones) left behind by others. Through these interactions, the exploration of the simulation space and the optimization of various tasks are achieved.

Polygon Image Recreation Algorithm

https://github.com/Teppi-txt/Polygon-Image-Recreation

A program which recreates existing artworks or photographs using basic polygons using an evolution algorithm to select the shape that best fits the image, inspired by a previous procedural generation algorithm for terrain. This project was made over the summer of 2022 to practice Java development.

Workflow Assistant

https://chrome.google.com/webstore/detail/workflow-assistant

An extension on the chrome web store which helps struggling students (like me) with theit time management and be more productive while studying or doing assessments. Features a site blocker, simple todo list, and a work-break timer.
